About Virtual Clinic

Virtual Clinic is an application that can be used by both doctors and their patients.
It allows for either user to sign up and be given a role as a doctor or patient.
Patients can view the staff directory, their prescriptions, submit reports with their vitals and pain information, and schedule appointments with their doctor.
Doctors can view their patients, create prescriptions, view and add notes to patient reports, and set times in their calendar where an appointment can be created with them.

Technologies Used

  • C#
  • ADO.NET Entity Framework
  • ASP.NET Web API
  • Angular
  • TypeScript
  • Microsoft Azure
  • DevOps
  • JavaScript
  • CSS
